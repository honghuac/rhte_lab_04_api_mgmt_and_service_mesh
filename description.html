API Management and Service Mesh

API Management and Service Mesh are complimentary technologies. This multi-product lab guides you through an integration between the two technologies and also provides a comparison and contract between them.

The lab is deployed on Red Hat OpenShift Container Platform and includes the following components:
  • Red Hat 3scale API Management
  • Red Hat OpenShift Application Runtimes
  • community Istio
